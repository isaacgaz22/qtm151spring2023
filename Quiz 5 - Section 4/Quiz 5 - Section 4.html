(e) Aggregate dataset by time period

  • Create an object "month_config" that uses pd.Grouper(), setting the key to

the date column and freq to "m" (month)

  • Follow the instructions in Lecture 21 to compute an aggregate dataset

by month which computes the standard deviation of "cb_to_usd"

  • Display the dataset

# Write your own code

month_config = pd.Grouper(key='date', freq='m')

monthlydata = (bitcoin
                .groupby(month_config)
                .agg(cb_to_usd_std = ("cb_to_usd","std"))
                .reset_index()
                )
